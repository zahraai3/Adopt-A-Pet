# React Animals Project

This project is a simple React app built to practice **React Routing** and working with multiple pages.  

## Overview

- The app displays different **animals** and their **details**.  
- Animals are categorized by type.  
- Simple navigation between pages using **React Router**.  
- Focused on learning **React basics** and routing, without extra features or complexity.

## Features

- Multiple pages for different animal categories.  
- Detail pages for each animal.  
- Easy-to-use navigation.  

## Technologies Used

- **React**  
- **React Router**
